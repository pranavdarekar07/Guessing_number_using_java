
A simple console-based Java game where the player has 5 chances to guess a randomly generated number between 1 and 100. The program provides hints ("Too high" / "Too low") after each attempt and validates user input to ensure only positive integers are accepted.

## ✨ Features
- Random number generation between 1–100
- Input validation for non-integer and out-of-range values
- Limited attempts (5 chances per game)
- Clear feedback messages for each guess
- Win/lose conditions with final reveal of the number
